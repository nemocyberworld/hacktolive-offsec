URLextractor is a shell-based reconnaissance tool created by eschultze, included in BlackArch Linux. It's designed for passive recon and URL analysis.

1. About the Tool

##################################################
#                  URLextractor                  #
# Information Gathering & Website Reconnaissance #
#              coded by eschultze                #
#           https://phishstats.info/             #
#                version - 0.2.0                 #
##################################################

Purpose: Extracts useful information from a target URL using passive methods.

2. Usage

./extractor http://site.com/
./extractor http://site.com/login/
./extractor http://site.com/proxy.pac

If you run it without arguments:

[ALERT] NO target set
[ALERT] USAGE: ./extractor http://site.com/ OR http://site.com/path/dir/file.php

3. What It Extracts

  • IP address resolution
  • GeoIP (region, country, ISP)
  • SSL Certificate Info
  • Subdomain detection
  • robots.txt or other exposed endpoints
  • MD5 of content for comparison
  • Abuse contact details

4. Complement with Web Parser

The terminal output can be overwhelming. Use a web parser (browser-based) to extract structured fields like:

  • Target hostname and IP
  • Country, ASN, ISP
  • Certificate issuer
  • Subdomains
  • Found links and pages

5. Example Bash Script to Save Outputs

#!/bin/bash

TARGET=$1
OUTFILE=$(echo "$TARGET" | sed 's|https\?://||; s|/|_|g')
./urlextractor "$TARGET" | tee "${OUTFILE}_recon.txt"
