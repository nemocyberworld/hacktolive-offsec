Domain: Lab • Focus: Nmap Advanced • Last Updated: 2025-08-31

Summary

First lab task: set up Metasploitable3 (Ubuntu 14.04 OVA) in VirtualBox, configure network (Bridge mode), and practice advanced Nmap scans. Everyone should share notes and build their own resources — no repetition later.

Steps

  1. Download OVA: Metasploitable3-ub1404
  2. Import into VirtualBox.
  3. Set Network: Adapter-1 Bridge, Adapter-2 Bridge.
  4. Start VM, confirm connectivity.
  5. Run Nmap scans (ports, services, versions, vuln scripts).
  6. Share notes/resources in Telegram channel.

Commands

# Basic
nmap -sS -p- 192.168.1.100

# Service detection
nmap -sV -sC 192.168.1.100

# Aggressive scan
nmap -A 192.168.1.100
