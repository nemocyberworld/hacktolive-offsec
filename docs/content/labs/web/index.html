Web Penetration Testing Labs

Domain: Web • Focus: Hands-on Web App Security Practice • Last Updated: 2025-09-05

Summary

A curated list of free (and freemium) web security practice labs: intentionally vulnerable applications, guided academies, and self-hosted targets. Use these to practice discovery, exploitation, and remediation of common issues (OWASP Top 10, authz/authn flaws, injections, deserialization, SSRF, IDOR, API/GraphQL vulns, secrets handling, and more) in a safe, legal environment.

Available Labs

  • OWASP Broken Web Apps – VM bundling many classic vulnerable apps.
  • OWASP Juice Shop – Modern full-stack app with dozens of vulnerabilities.
  • bWAPP – Intentionally buggy PHP app covering a wide range of vulns.
  • DVWA – Damn Vulnerable Web Application for fundamentals.
  • Mutillidae II – OWASP Top 10 practice with hints and training materials.
  • WebGoat – OWASP’s lesson-driven vulnerable app.
  • XVWA – Xtreme Vulnerable Web App with diverse scenarios.
  • Vulnerable Docker Labs – Collection of dockerized vulnerable apps.
  • Security Shepherd – OWASP training platform with guided challenges.
  • PortSwigger Web Security Academy – Free interactive labs (beginner → expert).
  • Hack.me – Hosted demo vulns and community scenarios.
  • DefendTheWeb – Browser-based web challenge platform.
  • Game of Hacks – Quick quiz-style challenges on common flaws.
  • Google Gruyere – Tiny but rich lab for classic web vulns.
  • XSS Game – Google’s browser-based XSS challenges.
  • Testfire Demo – “Altoro Mutual” demo banking site.
  • Acunetix Test Site – Vulnerable PHP site for scanners/manual testing.
  • Zero Bank Test Site – Deliberately insecure banking app.
  • Hacksplaining Labs – Bite-sized interactive exercises with explanations.
  • PentestIT Lab – Remote vulnerable infrastructure (availability varies).
  • Labs Wizard Security – Community web challenge sets.
  • OWASP NodeGoat – Vulnerable Node.js/Express app aligned to OWASP risks.
  • OWASP RailsGoat – Deliberately insecure Ruby on Rails app.
  • OWASP crAPI – “Completely Ridiculous API”: end-to-end vulnerable API ecosystem.
  • DVGA – Damn Vulnerable GraphQL Application.
  • VAmPI – Vulnerable REST API for learning API security issues.
  • DVWS – Damn Vulnerable Web Services (SOAP/REST) for API/Web-services testing.
  • OWASP WrongSecrets – Learn secrets management pitfalls in apps/cloud.
  • DVNA – Damn Vulnerable Node Application (Node.js).
  • Broken Crystals – Web app with intentional crypto/design flaws.
  • SQLi Labs – Focused labs for SQL injection techniques.
  • BodgeIt (archived) – Older Java shop app; still useful for legacy flaw patterns.
  • WebGoat.NET (archived) – .NET variant for Windows/.NET testing.
  • Damn Vulnerable Web Sockets – Practice WebSocket-specific vulns.
  • OWASP VulnerableApp – Launcher hosting multiple vulnerable apps and lessons.
  • Web Security Dojo – Ready-to-use VM preloaded with tools & vulnerable apps.
