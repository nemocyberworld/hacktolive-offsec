Network Penetration Testing Labs

Domain: Network • Focus: Hands-on Network Exploitation & Enumeration Practice • Last Updated: 2025-09-05

Summary

This collection focuses on deliberately vulnerable machines and environments for practicing network reconnaissance, service enumeration, exploitation, privilege escalation, and post-exploitation. Most are lightweight VMs or ready-to-run images suitable for home labs and classroom use. Use them to build repeatable workflows with tools like Nmap, Netcat, Metasploit, and custom exploits.

Available Labs

  • Metasploitable 2 – Classic intentionally vulnerable Linux VM; great for enumeration & service exploitation.
  • Metasploitable 3 – Vulnerable Windows & Linux builds for end-to-end exploitation practice.
  • Damn Vulnerable Linux (DVL) – Linux distro packed with misconfigurations and vulnerable services.
  • Kioptrix Series – Progressive vulnerable VMs (Level 1–5) focused on classic network vulns.
  • DC Labs – DC-1 to DC-9: Linux VMs emphasizing web-to-shell and lateral movement.
  • Stapler – Enumeration-heavy Linux target; great for practicing service discovery paths.
  • Mr. Robot VM – Mr. Robot-themed Linux machine; pivot from web footholds to root.
  • Hackademic RTB1 – Web app to network exploitation chain; beginner-friendly path to privilege escalation.
  • LampSecurity VMs – Multiple vulnerable LAMP stacks for service and web exploitation.
  • Bastard VM – Windows-leaning target; OSINT + enumeration + privesc chain practice.
  • Morning Catch – Prebuilt lab scenario for client-side/network attack flows.
  • Moth VM – Vulnerable services & scripts bundled for quick exploitation drills.
  • VPL-Environment – One-stop VM that aggregates several vulnerable apps and services.
  • Security Onion – Blue-team distro for NSM/IDS; mirror attacker traffic and analyze PCAPs.
  • Blue Team Labs Online – DFIR/SOC challenge scenarios; good for seeing your attacks from the defender’s view.
  • Penetration Testing Practice Labs (Aman Hardikar) – Massive curated index of vulnerable machines and practice targets.
