Cloud Labs

Domain: Cloud • Focus: Cloud Security & Exploitation Training • Last Updated: 2025-09-05

Summary

Cloud labs focus on helping learners understand vulnerabilities, misconfigurations, and attack vectors in cloud environments. These platforms provide hands-on challenges to explore AWS, GCP, Kubernetes, and general cloud exploitation scenarios. They are excellent for developing both offensive pentesting skills and defensive cloud security practices.

Available Labs

  • Flaws.cloud – Free AWS-based lab designed to teach common cloud security flaws and misconfigurations.
  • Flaws2.cloud – Sequel to Flaws.cloud with advanced AWS exploitation challenges.
  • Thunder CTF – Free cloud-focused capture the flag with infrastructure-as-code challenges and real misconfigurations.
  • Google CTF (Cloud Challenges) – Google’s annual CTF event includes cloud exploitation scenarios accessible for free.
  • Cybr Free AWS Labs – 20+ free browser-based AWS security labs covering IAM, S3, Lambda, and more.
  • Pentester Academy AWS Labs – Cloud security labs focusing on IAM misconfigurations and AWS exploitation.
  • Awesome CloudSec Labs – Curated list of cloud-native security CTFs, workshops, and self-hosted labs across AWS, GCP, and Kubernetes.
  • AWS Well-Architected Security Labs – Guided labs teaching AWS security best practices and misconfiguration remediation.
