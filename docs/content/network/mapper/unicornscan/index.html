Unicornscan

Domain: Network • Focus: High-Performance Asynchronous Port Scanning • Last Updated: 2025-09-04

Summary

Unicornscan is an advanced information gathering and port scanning tool designed for large-scale network reconnaissance. Unlike traditional scanners, it leverages asynchronous TCP and UDP scanning techniques to achieve higher performance and stealth. Unicornscan is often used in penetration testing, red teaming, and research to discover open services, analyze banners, and gather intelligence on large address spaces. It provides flexible packet crafting, fine-grained control over timing, and supports exporting results for further analysis.

Steps

  1. Identify the target host(s) or network range you want to scan.
  2. Select the scan type (TCP, UDP, service discovery) and set timing parameters.
  3. Run Unicornscan with the chosen flags to perform reconnaissance.
  4. Interpret results (open ports, service banners) and log or export findings for reporting or exploitation.

Commands


# TCP Scan
unicornscan -Iv -p 80,443 192.168.1.10       # TCP SYN scan on ports 80,443

# UDP Scan
unicornscan -mU -Iv -p 53,161 192.168.1.10   # UDP scan on DNS and SNMP ports

# Range Scan
unicornscan -Iv -p 1-1000 10.0.0.0/24        # Scan first 1000 ports on /24 network

# Service Detection
unicornscan -mT -Iv -p 22,25,80 target.com   # Grab service banners on selected ports

# Output to File
unicornscan -mT -Iv -p 1-65535 192.168.1.5 -w results.txt   # Save full scan results

# Example Usage
unicornscan -Iv -p 80 192.168.1.1           # Fast single-port TCP scan
unicornscan -mU -p 123 192.168.1.100        # NTP (UDP) scan on specific host

Best Flag Combinations (Cheat Ready)


# Quick TCP scan (common web ports)
unicornscan -mT -Iv -p 80,443 192.168.1.10        # Fast TCP SYN scan on web ports

# Full range scan on subnet
unicornscan -mT -Iv -p 1-65535 10.0.0.0/24        # Complete port sweep on /24 range

# UDP service discovery
unicornscan -mU -Iv -p 53,161,123 192.168.1.50    # Check for DNS, SNMP, NTP

# Banner grabbing (fingerprint services)
unicornscan -mT -Iv -p 21,22,25,80 target.com     # Collect banners from common services

# Stealthier scan with low rate
unicornscan -mT -r50 -Iv -p 1-1000 192.168.1.20   # Slow rate scan to avoid detection

# Output to file for reporting
unicornscan -mT -Iv -p 1-65535 192.168.1.30 -w report.txt   # Save results to file

# Dual-stack (IPv6 scanning)
unicornscan -mT -Iv -p 22,80,443 2001:db8::1               # Scan IPv6 host on key ports

# Compare TCP vs UDP on same target
unicornscan -mT -Iv -p 80,443 192.168.1.5                  # TCP SYN scan
unicornscan -mU -Iv -p 80,443 192.168.1.5                  # UDP scan for service response
