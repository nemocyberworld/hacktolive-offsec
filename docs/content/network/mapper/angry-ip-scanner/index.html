Summary

Angry IP Scanner (ipscan) is a lightweight, open-source, cross-platform GUI network scanner. It allows rapid detection of live hosts, open ports, hostnames, MAC addresses, and NetBIOS data and provides export options like TXT, CSV, and XML. Ideal for quick net recon in pentesting or sysadmin workflows.

Steps

  1. Download Linux package from the official site: Linux Downloads (DEB, RPM, JAR) .
  2. Install via your package method (DEB/RPM) or run the JAR directly (requires Java 17+).
  3. Launch with ipscan from terminal or app menu.
  4. Drag your mouse to enter an IP, hostname, or range (e.g. 192.168.1.0â€“255).
  5. Select scan options (ping, ports, NetBIOS, hostname resolution, MAC).
  6. Click **Start** â†’ review results in the GUI table.
  7. Export results via **File â†’ Save As** to TXT, CSV, or XML.
