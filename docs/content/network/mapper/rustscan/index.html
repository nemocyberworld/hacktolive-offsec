RustScan

Domain: Network Scanning • Focus: Fast Port Scanning & Nmap Integration • Last Updated: 2025-09-04

Summary

RustScan is a modern, lightning-fast port scanner written in Rust. It is designed to scan targets extremely quickly by leveraging async operations, then hand over discovered open ports to Nmap for detailed service and vulnerability analysis. It reduces scan times drastically compared to traditional scanners, making it especially useful during penetration testing and reconnaissance.

Steps

  1. Identify the target host(s) or IP ranges you want to scan.
  2. Select RustScan flags for speed, scripting, or Nmap integration.
  3. Run RustScan to enumerate open ports quickly.
  4. Pipe results into Nmap for service enumeration or vulnerability scanning.

Commands


# Target & Input
rustscan -a 192.168.1.10                        # --addresses  Scan a host or IP
rustscan -a 192.168.1.10,10.0.0.5,scanme.nmap.org # Multiple targets comma-separated
rustscan -a targets.txt                         # Supply file with newline-separated hosts

# Port Selection
rustscan -p 80,443,8080 192.168.1.10            # --ports  Scan specific ports
rustscan -r 1-1000 192.168.1.10                 # --range  Scan a range of ports
rustscan --top -a 192.168.1.10                  # --top    Scan top 1000 ports
rustscan -e 22,25,110 -a 192.168.1.10           # --exclude-ports  Skip selected ports

# Address Exclusion
rustscan -x 10.0.0.0/24 -a hosts.txt            # --exclude-addresses  Skip CIDR/IPs/hosts from scan

# Output & Modes
rustscan -g -a 192.168.1.10                     # --greppable  Output only open ports (grep-friendly)
rustscan --no-banner -a 192.168.1.10            # Hide ASCII banner
rustscan --accessible -a 192.168.1.10           # Accessible mode (better for screen readers)

# Config & Resolver
rustscan -n -a 192.168.1.10                     # --no-config Ignore config file
rustscan -c /tmp/custom-config.toml -a 192.168.1.10  # Use custom config path
rustscan --resolver 1.1.1.1,8.8.8.8 -a domain.com   # Custom DNS resolvers list or file

# Speed & Performance
rustscan -b 1000 -a 192.168.1.10                # --batch-size Adjust probes per second (default 4500)
rustscan -t 2000 -a 192.168.1.10                # --timeout Timeout in ms (default 1500)
rustscan --tries 3 -a 192.168.1.10              # --tries Number of attempts per port
rustscan -u 65535 -a 192.168.1.10               # --ulimit Raise file descriptor limit
rustscan --scan-order random -a 192.168.1.10    # --scan-order [serial|random] port scanning order

# Scripts & Service Analysis
rustscan --scripts none -a 192.168.1.10         # Disable extra scripts
rustscan --scripts custom -a 192.168.1.10       # Use custom scripts if available
rustscan -- -sV -A -O                           # Pass open ports directly to Nmap (service + OS detection)

# Protocols
rustscan --udp -a 192.168.1.10                  # UDP mode (detect UDP ports with responses)

# Info & Help
rustscan -h                                     # --help Print help
rustscan -V                                     # --version Show RustScan version

# Example Usage
rustscan -a 192.168.1.10 -- -sV                 # Fast scan + service detection via Nmap
rustscan -a scanme.nmap.org -r 1-5000 -- -A     # Scan first 5000 ports and run aggressive Nmap scan

Best Flag Combinations (Cheat Ready)


# Fast all-ports sweep with Nmap service detection
rustscan -a 192.168.1.10 -r 1-65535 -- -sV

# Quick top 1000 ports + aggressive Nmap follow-up
rustscan -a scanme.nmap.org --top -- -A

# Multiple hosts with randomized scan order
rustscan -a hosts.txt --scan-order random -b 2000 -- -sC -sV

# UDP scan on common ports
rustscan -a 192.168.1.20 --udp -p 53,123,161 -- -sU

# Subnet-wide sweep with grep-friendly output
rustscan -a 192.168.1.0/24 -g -b 1000

# Exclude noisy ports (22,25,110) from scan
rustscan -a 192.168.1.10 -r 1-1000 -e 22,25,110

# Raise ulimit for massive scan & speed up
rustscan -a 10.0.0.1 -r 1-65535 -u 65535 -b 5000 -- -sV

# Passive mode to only print open ports
rustscan -a example.com -g

# Scan with custom DNS resolvers (Cloudflare + Google)
rustscan -a domain.com --resolver 1.1.1.1,8.8.8.8 -- -sV

# Save results with Nmap output to XML
rustscan -a 192.168.1.15 -r 1-5000 -- -sV -oX results.xml
