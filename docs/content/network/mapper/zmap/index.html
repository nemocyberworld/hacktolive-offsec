ZMap

Domain: Network Scanning • Focus: Internet-wide Port Scanning • Last Updated: 2025-09-04

Summary

ZMap is an open-source network scanner designed for Internet-wide studies. It enables researchers, penetration testers, and defenders to rapidly scan the entire IPv4 address space for open ports and services. Known for its extremely fast performance compared to traditional scanners like Nmap, ZMap supports selective scanning, rate limiting, blacklisting, and modular probes for protocols such as TCP SYN, UDP, ICMP, and custom probes. It is commonly used in security research, vulnerability assessments, and measurement of Internet services (e.g., HTTPS, SSH, DNS).

Steps

  1. Install ZMap from source or package manager (Linux recommended).
  2. Select the probe module (e.g., TCP SYN) and define the port or service to scan.
  3. Set scan parameters like rate limit, blacklist/whitelist, and output format.
  4. Run the scan and save results to file (JSON/CSV compatible with analysis tools).
  5. Analyze discovered hosts and services for research or penetration testing purposes.

Commands


# Basic TCP SYN scan on port 80
zmap -p 80 0.0.0.0/0 -o results.csv

# Scan with rate limit (1000 packets per second)
zmap -p 22 0.0.0.0/0 --rate=1000 -o ssh_hosts.csv

# Scan a specific subnet only
zmap -p 443 192.168.1.0/24 -o https_hosts.json -f json

# Use blacklist to exclude sensitive IP ranges
zmap -p 21 0.0.0.0/0 --blacklist-file=blacklist.conf -o ftp_hosts.csv

# Run with ICMP echo probe instead of TCP SYN
zmap --probe-module=icmp_echoscan -o icmp_alive.csv

# Example Usage
zmap -p 80 203.0.113.0/24 -o webscan.csv    # Scan /24 subnet for web servers
zmap -p 25 0.0.0.0/0 --rate=5000 -o smtp.csv # Full IPv4 scan for SMTP (port 25)

Best Flag Combinations (Cheat Ready)


# Fast web server sweep across the Internet
zmap -p 80 0.0.0.0/0 --rate=100000 -o web_servers.csv

# SSH host discovery with reduced scan rate
zmap -p 22 0.0.0.0/0 --rate=500 --cooldown-time=60 -o ssh_hosts.csv

# Subnet-specific HTTPS scan with JSON output
zmap -p 443 192.168.1.0/24 -o https.json -f json

# Exclude sensitive IP ranges during scan
zmap -p 25 0.0.0.0/0 --blacklist-file=blacklist.conf -o smtp_hosts.csv

# ICMP ping sweep (alive hosts only)
zmap --probe-module=icmp_echoscan --rate=20000 0.0.0.0/0 -o alive_hosts.csv

# Custom source IP & MAC for evasion
zmap -p 80 203.0.113.0/24 --source-ip=198.51.100.10 --source-mac=00:11:22:33:44:55 -o scan.csv

# Service-specific scanning with whitelist
zmap -p 21 --whitelist-file=targets.txt --rate=2000 -o ftp_results.csv

# Multi-port scan with output filtering
zmap -p 21,22,80,443 0.0.0.0/0 --rate=5000 -o multiport.csv
